Project Idea

The Hackathon Rehearsal Project is a practice project created to help our team understand the complete software development workflow before participating in the CODEZAP25 Hackathon.

The project focuses on learning:

Git and GitHub collaboration
Team-based development
Version control
Repository management
Task allocation
README documentation
Commit history
Team coordination

This rehearsal helps ensure every team member understands how to contribute efficiently during the actual hackathon.

What We Learned

During this rehearsal, our team learned how to:

Create a GitHub repository.
Initialize a Git repository using git init.
Stage files using git add.
Commit changes with meaningful commit messages.
Push projects to GitHub.
Collaborate using GitHub repositories.
Assign roles and responsibilities within the team.
Maintain project documentation using a README file.

Our goal is to build a high-quality project by combining our skills in frontend, backend, and project management while following industry-standard Git and GitHub workflows. This rehearsal prepares us to collaborate effectively and deliver our project successfully during CODEZAP25.
